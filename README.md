# QuickNote
A Simple Mobile App for quick adding notes developed With .NET MAUI and local sqlite database.
Tested on Android 12.

# Packages Used:

CommunityToolkit.Mvvm

sqlite-net-pcl

SQLitePCLRaw.bundle_green
